# MATLAB
MATLAB cheat sheet

## Get help
```MATLAB
help [command]
```
## Useful commands
* List all variables :
>```MATLAB
>whos
>```
* Clear all variables :
>```MATLAB
>clear
>```
* Clear terminal :
>```MATLAB
>clc
>```
* Squareroot :
>```MATLAB
>sqrt(2)
>```
* Module (complex) :
>```MATLAB
>abs(5+4i)
>```
* Number of lines and columns :
>```MATLAB
>size(matrice)
>```


## Variables :
* Matrix:
```matlab
matrice=[1, 2; 3, 4]
```
Output :
```
matrice =

     1     2
     3     4
```
* Complex number:
```matlab
z=4+5i
```
Output :
```
z =

   4.0000 + 5.0000i
```
